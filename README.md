## Overview

A tool that anyone can use to reach their fitness goals more quickly. Use to track workout progress. This assignment will requires a Mongo database with a Mongoose schema and handle routes with Express.

## Technology Used
- MongoDB
- Express
- Mongoose
- Morgan
